## 📈 What Happens When You Send a Request

```
You (Postman/cURL)
    ↓
HTTP Request → Flask Server (Port 5000)
    ↓
Request Validation
    ↓
Call CustomerCommunicatorAgent
    ├─ Message Generator Agent → LLM
    └─ Compliance Validator Agent → LLM
    ↓
Generate Response
    ↓
Return JSON to Postman
    ↓
You See Response
Terminal Shows Logs
```
