## рҹҡҖ 5 API Endpoints

### 1. Health Check
```
GET /health
```
Quick API availability check.

### 2. Agent Status
```
GET /api/v1/status
```
Get agent configuration and endpoints.

### 3. Generate Single Message вӯҗ
```
POST /api/v1/generate-message
```
Generate one personalized customer message.
- Input: customer_profile, resolution_plan, credit_confirmation
- Output: Personalized message + compliance validation

### 4. Batch Generate Messages вӯҗ
```
POST /api/v1/batch-generate
```
Generate multiple customer messages.
- Input: Array of message data
- Output: Results array with status per message

### 5. Validate Message
```
POST /api/v1/validate
```
Validate message for GDPR and brand compliance.
- Input: Message content + context
- Output: Compliance status + issues + suggestions

## рҹ“һ Troubleshooting

### Port 5000 Already in Use
```bash
# Find and kill process
lsof -ti:5000 | xargs kill -9
```

### Flask Not Installed
```bash
python -m pip install flask flask-cors
```

### Agent Not Initializing
- Verify `.env` has valid Azure credentials
- Check `OPENAI_API_KEY` is set
- Run: `python customer_communicator_agent.py` first

### Requests Timeout
- Increase Postman timeout: Settings вҶ’ General
- Check internet connection
- Verify Azure OpenAI service status
